# k8s_PaaS

如何基于 K8S 部署成 PaaS（一套完整的软件研发和部署平台）—— 教程 / 学习（实战代码 / 欢迎讨论 / 大量注释 / 操作配图），你将习得部署如：K8S、dashboard、Harbor、Jenkins、本地 gitlab、Apollo 框架、promtheus、grafana、spinnaker 等。

注释及配图覆盖率达 80% 以上，旨在帮助快速入门。

并将告诉你：是什么（WHAT）、为什么这么做 (WHY)、怎么做 (HOW)。

建议学习时长 1 个月 +，最终将实现点点点（自动化）的形式就能部署上线并维护。
