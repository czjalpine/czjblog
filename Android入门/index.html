# Android 入门

# 布局管理器

# 一、线性布局（LinearLayout）

# 属性

  • android:orientation （专属）

    • vertical—— 垂直
    • horizontal—— 横向

  • android:gravity （子元素的对齐方式）

  • android:layout_width 和 android:layout_height

    • match_parent—— 匹配父空间
    • wrap_content—— 包含自身内容
    • 像素点单位用 dp，字体使用 sp

# 内容平分

在子元素中加入 android:layout_weight

当其中元素的大小都为 0 时，开始根据权重分配父元素空间；

当其中元素大小存在数额时，先把数额划分出来。多出的部分再用权重划分

# 二、相对布局（RelativeLayout）

# 属性

  • android:layout_ toLeftOf
  • android:layout_ below
  • android:layout_ toRightOf
  • android:layout_ alignBottom
  • android:layout_ alignParentBottom

跟 TextView 控件一样设置，但 20.3.1 版本可能会出现一下情况
